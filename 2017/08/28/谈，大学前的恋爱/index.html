谈，大学前的恋爱

2017/8/28 随心写 心情 感想

人生，终究逃不过一个“情”字。

沉鱼落雁，沦落成美。

在最美的年华，难免会做几件疯狂的事，不是无知，而是感性。在我看来，这样的认知也并不算一件坏事，但在步入大学之前，谈什么天长地久，天涯海角，是完全弄错了“爱情”这个词的真正蕴含意义。

我不懂爱情，也不想去彻底弄懂它。

多少个山盟海誓，幻化成昨日的泡影，所以才有了一步一步的摧残。尚在高中就读，自嘲能对这么深远的一词谈论一番。如果你能试着说着别人说过的话，便知道其中的真正用意。我不是没有见过，在这样的阶段，许多同龄人都掩饰不住自己的那青春之心，这十分正常。但这个过程的目的并不是去牵想牵的手，吻想吻的脸，这就十分无趣了，明显走歪了航道。于我所言，异性相知，心有灵犀，即可。不可多求，不可多想。就是所谓的，在学途中，于孤独中寻求倾诉，于欢乐中寻求共享，于风雨中寻求共苦。相遇中相知，这就是在步入大学前所有恋爱的最好诠释。

作为普通人的我也是想的。不敢多想，却又难免不想。秋风一阵阵拂过，带来一幕幕的相思。我不敢继续往前走，因为这个世界已经不再单纯地相信了，太多的误解，蒙住了一双双稚嫩的眼睛。于是被放逐，在众多的理由前被迫退缩。我，便只有期待，去空忆。在某一天的清晨，自己对自己说。

远处的树绿，近处的花红，偶尔天边几处黑影掠过。谁的身影走过，却没有留下一声招呼，更别说再见了。

我自嘲自笑，自己竟写出这些话语。

说真的，我不是太懂“爱情”这词，只知道给人带来温暖，而不是随意的嘴边话语，也不是所谓的个人私欲。

我，只有乱猜测罢了。

聂振杰
